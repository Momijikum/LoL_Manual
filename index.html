LoLってどんなゲーム？

リーグオブレジェンド（LoL）は、マルチプレイヤー・オンライン・バトル・アリーナ（MOBA）のゲームです。

約170体のチャンピオンと、それぞれのスキル・ストーリーが魅力です。毎試合が異なるため戦略とプレイスキルの両方が重要です。

ルール

舞台はサモナーズリフト。基本はネクサスを破壊した方の勝利です。

  • ネクサス → タワー → インヒビター → タワー → タワー → 勝利
  • 必要なもの：金・経験・バフ

ロール

  • TOP: タイマン担当、一人で試合を破壊も可能
  • JG: 森にいて全レーンに関与、難易度高め
  • MID: 影響力が高い花形レーン
  • ADC: DPS火力担当、狙われやすい
  • SUP: サポート、味方を守る

試合の流れ

  1. チャンピオンをバン
  2. チャンピオンをピック
  3. ルーン設定
  4. サモナースペル設定

スキルはクリックまたはCtrl + スキルでレベルアップ。6, 11, 18でRスキル強化。

覚えておきたい知識

  • フラッシュ: 壁を越えたり逃げ攻めに使用
  • スマイト: モンスターに大ダメージ、JG専用
  • ノックアップ: 浮かされ何もできない
  • スロウ: 足が遅くなる

ステータスを上げたり特殊効果を得られます。素材から買いましょう。金色の枠のアイテムは特に注意。

  • キーストーン例：電撃、エアリー、魂の収穫、ファーストストライク
